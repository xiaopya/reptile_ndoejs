##### 1. 下载并安装依赖

> 前提你得有node

```shell
git clone https://github.com/xiaopya/reptile_ndoejs.git

cd reptile_ndoejs/

yarn 
or
npm install
```

##### 2. 安装好依赖之后 就可以运行

```shell
# 执行以下命令
node app.js
```
